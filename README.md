# RaceSimulation

A Java object-oriented horse racing simulator with both textual and GUI-based race visualisation.

This project was developed as part of an Object-Oriented Programming module and focuses on class design, simulation logic, GUI interaction, and configurable race settings.

## Features

- Text-based race simulation
- GUI race visualisation
- Horse class with configurable name, symbol, confidence, and status
- Race class for managing lanes, race distance, and winner detection
- Randomised race progress and fall logic
- Separate Part 1 and Part 2 implementations

## Tech Stack

- Java
- Java Swing / GUI components
- Object-Oriented Programming

## Project Structure

```text
Part1/  Text-based race simulation
Part2/  GUI race simulation
```

## How to Run

### Text Version

```bash
cd Part1
javac *.java
java startRace
```

### GUI Version

```bash
cd Part2
javac *.java
java startRaceGUI
```
